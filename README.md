# Fuelytics 🚗⛽

Fuelytics es una aplicación web full-stack diseñada para registrar, analizar y optimizar el consumo de combustible y los gastos asociados a vehículos.


## 🎯 Objetivo del proyecto

El objetivo de Fuelytics es desarrollar una aplicación completa que permita a los usuarios gestionar sus vehículos, registrar repostajes y gastos, y obtener estadísticas reales de consumo y costes.

Este proyecto se ha realizado como parte del ciclo superior de Desarrollo de Aplicaciones Multiplataforma (DAM), aplicando buenas prácticas de desarrollo y control de versiones.

## 🚀 Funcionalidades principales

- Registro de usuarios
- Gestión de vehículos por usuario
- Registro de repostajes
- Cálculo de consumo medio
- Registro de gastos (mantenimiento, seguros, reparaciones, etc.)
- Estadísticas de consumo y costes
- Sistema de privacidad por vehículo

## 🛠 Tecnologías utilizadas

- Frontend: React
- Backend: Spring Boot
- Base de datos: MySQL / PostgreSQL
- Control de versiones: Git + GitHub

## 🗄️ Modelo de base de datos

El sistema está compuesto por las siguientes entidades principales:

- users
- vehicles
- fuel_entries
- expenses

Incluye relaciones entre usuarios y vehículos, así como registros de consumo y gastos asociados a cada vehículo.


## 🧱 Arquitectura del proyecto

El proyecto sigue una arquitectura cliente-servidor:

- Frontend: interfaz de usuario desarrollada en React
- Backend: API REST desarrollada con Spring Boot
- Base de datos relacional
